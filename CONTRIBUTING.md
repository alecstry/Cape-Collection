# 🤝 Contribuir

Gracias por tu interés en mejorar **Cape Collection**.

Este proyecto está pensado para la comunidad de Minecraft Bedrock, creadores de contenido y jugadores que quieran ayudar a mejorar la colección con reportes, sugerencias o nuevas ideas.

---

## ✨ Puedes contribuir con

- 🛠️ Reportes de errores.
- 🧭 Sugerencias de organización.
- 🎨 Propuestas de nuevas capas.
- 📸 Capturas de prueba.
- 📝 Mejoras de documentación.
- 🌐 Traducciones o correcciones de texto.

---

## 🔎 Antes de abrir un issue

Revisa si el problema está relacionado con:

- 🧍 Otro resource pack que modifica `player.entity.json`.
- 🪽 Otro pack que modifica Elytra.
- 🖼️ Una textura mal nombrada.
- 📁 Un subpack que no coincide con el `folder_name` del `manifest.json`.
- 🎮 Una versión incompatible de Minecraft Bedrock.

---

## 🛠️ Formato sugerido para reportar errores

```md
### Problema
Describe qué sucede.

### Versión de Minecraft Bedrock
Ejemplo: 1.26.x / 26.x

### Plataforma
Windows / Android / iOS

### Pasos
1. Activé el pack.
2. Seleccioné la capa.
3. Entré al mundo.
4. Ocurrió el error.

### Capturas o logs
Adjunta imágenes si es posible.
```

---

## 🎨 Propuestas de nuevas capas

Para sugerir una capa, incluye:

- 🏷️ Nombre sugerido.
- 📚 Si es oficial, histórica, experimental o personalizada.
- 🖼️ Imagen de referencia.
- 💡 Motivo para incluirla.
- 🪽 Si debería tener versión Elytra.
