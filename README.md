# Cape Collection

**Cape Collection** es un pack de recursos para **Minecraft Bedrock Edition** que permite visualizar una colección de capas mediante subpacks, con soporte visual para Elytra.

Este pack está pensado para pruebas visuales, showcases, reviews y creación de contenido relacionado con capas de Minecraft. Su objetivo es facilitar la visualización de diferentes diseños dentro del juego sin modificar bloques, ítems, mobs, biomas ni mecánicas.

> ⚠️ **Importante:** este pack no otorga capas oficiales a tu cuenta de Minecraft. Solo modifica la visualización mediante un resource pack.

---

## ✨ Características

- Colección de capas organizada por categorías.
- Soporte visual para Elytra sincronizada con la capa seleccionada.
- Selector de diseños mediante subpacks.
- No requiere behavior pack.
- No modifica gameplay, bloques, ítems, mobs ni biomas.
- Compatible con Minecraft Bedrock Edition en versiones recientes.
- Ideal para showcases, pruebas visuales, videos, comparativas y reviews.

## 🧩 Instalación

1. Descarga el archivo **CapeCollection.mcpack** desde [Releases](https://github.com/alecstry/Cape-Collection/releases). 
2. Ábrelo con **Minecraft Bedrock Edition**.
3. Espera a que el juego importe el pack.
4. Ve a **Ajustes > Recursos globales > Mis packs**.
5. Activa **Cape Collection**.
6. En el engranaje del pack, selecciona la capa desde el selector de subpacks.

> En algunas versiones de Minecraft Bedrock, el selector de subpacks puede aparecer bajo el nombre **“Resolución”**. Esto forma parte de la interfaz interna del juego.

---

## 🔎 ¿Cómo funciona?

**Cape Collection** usa un sistema de **subpacks**.  
Cada subpack reemplaza la textura principal de la capa y permite visualizar un diseño distinto dentro del juego.  
Además, el pack incluye soporte visual para **Elytra**, por lo que al equiparlas se mostrará una textura sincronizada con la capa seleccionada.

## ⚙️ Compatibilidad

Este pack puede entrar en conflicto con otros resource packs que modifiquen archivos relacionados con:

- Capas.
- Elytra.
- Render controllers.
- Geometrías del jugador.
- `player.entity.json`
- `elytra.json`

Si otro pack modifica los mismos archivos, Minecraft puede priorizar uno sobre otro según el orden de carga.

Su objetivo será ofrecer una versión más compacta, premium y fácil de recorrer, con las capas más representativas o mejor valoradas del proyecto.
